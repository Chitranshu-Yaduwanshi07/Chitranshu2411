# 🚀 About Me

Hi 👋 I'm **Chitranshu Yadav**, a passionate **Computer Science Engineering student** and an aspiring **Java Full Stack Developer** from India. I enjoy building scalable web applications using **Java, Spring Boot, React, and MySQL**, while continuously improving my problem-solving skills through Data Structures & Algorithms.

Currently, I'm building **CodeNova**, an AI-powered SaaS platform, and **Skill Sensei**, an intelligent Learning Management System. My interests include **Artificial Intelligence, Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), and Spring AI**, where I love integrating AI into real-world applications.

Alongside development, I'm learning **AWS, Docker, Jenkins, Linux, and CI/CD** to strengthen my DevOps skills and understand the complete software development lifecycle.

> 💡 *Code • Learn • Build • Repeat*
